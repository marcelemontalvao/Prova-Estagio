# Prova-Estagio
1. Baseado no Scrum Guide, qual o tempo máximo deveriam ter as seguintes cerimônias: 
  a. Daily: deve durar no máximo 15 minutos.
  b. Planning: deve durar no máximo 8 horas para a sprint de um mês.
  c. Review: No máximo 4 horas para uma sprint de um mês.
  d. Retrospectiva: No máximo 3h para uma sprint de um mês.

2. José iniciou sua jornada recentemente com git e não está conseguindo publicar seu arquivo README e a classe Product.js que acabou de criar. O erro do git dá a seguinte informação: 
  Untracked Files: 
  README
  Product.js 
Por que esse “erro” de “Untracked files” ocorreu e qual comando resolveria o problema? 
  Esse erro ocorreu por que houve uma alteração no código e ela não foi adicionada ao diretório. O comando que resolveria esse problema seria o git add . e logo depois um commit -m “Nome do Commit” 

3. O time de desenvolvimento analisou e descobriu que o projeto está gerando arquivos pesados e desnecessários para serem comitados no repositório git. Em qual arquivo você deve incluir a lista de extensões para que esses arquivos desnecessários sejam desconsiderados? 
  .gitignore

4. Com suas palavras, descreva as vantagens de utilizar o Scrum no seu projeto. 
   Com o Scrum é possível adaptar o projeto de acordo com as necessidades que vão surgindo ao decorrer do tempo. Além disso, com o Scrum é possível acompanhar o progresso das atividades de cada um do grupo e assim, torna possível também a entrega de feedbacks de maneira melhor e mais contínua. Tudo isso contribui para a melhora do projeto final.
